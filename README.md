# Hospital Data Analysis using SQL

## Project Overview
This project analyzes hospital data using SQL to understand patient count, medical expenses, doctor availability, and performance across departments & cities.

## Key Insights
- Total patients data extracted
- Top departments & hospitals by performance
- Highest medical expenses identified
- Average hospital stay duration calculated
- Monthly expense analysis performed

## SQL Concepts Used
- Aggregate Functions (SUM, AVG, COUNT)
- GROUP BY & ORDER BY
- LIMIT
- DATE functions (DATEDIFF, DATE_FORMAT)

## Files Included
- hospital_data_analysis.sql (SQL queries)
- Hospital Data Assignment.pdf (original work)
- (Optional) screenshots of query results

## Tools Used
- MySQL 
- Excel dataset for source data

## Note
To view the output, execute the SQL queries in any SQL editor.
